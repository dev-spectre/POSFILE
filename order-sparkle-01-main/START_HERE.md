## 🚀 How to Get Started

### Option A: Quick Start (30 minutes)

1. **Read:** [QUICK_START.md](QUICK_START.md) (5 min)
2. **Setup:**
   - Copy .env.example → .env
   - Edit MONGODB_URI in .env
   - Run: `npm install`
   - Start MongoDB
3. **Run:**
   - Terminal 1: `npm run server:dev`
   - Terminal 2: `npm run dev`
4. **Test:** Visit http://localhost:5173

**Result: Fully working POS system in 30 minutes!**
